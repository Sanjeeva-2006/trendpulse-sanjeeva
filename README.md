# trendpulse-sanjeeva

This project was built step by step as a complete data pipeline using real-time data from the HackerNews API.

The work was divided into four tasks:

- Task 1: Collected trending stories using API and saved them in JSON format
- Task 2: Cleaned the data using Pandas and removed invalid or duplicate entries
- Task 3: Analysed the data using Pandas and NumPy and added new useful columns
- Task 4: Created visualizations and a dashboard using Matplotlib

Through this project, I understood how raw data is collected, cleaned, analysed, and finally visualized to extract meaningful insights.

I also learned how to handle real-world data issues like missing values, incorrect data types, and duplicates.

This project gave me a clear idea of how a basic data pipeline works from start to end.
